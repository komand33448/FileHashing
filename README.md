## FileHashing
C# .NET program that checks if hashes match

## Instalation


#### GNU+Linux
clone the git
~~~
$ git clone https://github.com/komand33448/FileHashing-Beta/
~~~

cd into the git
~~~
$ cd FileHashing-Beta
~~~

Run the software
~~~
$ dotnet run Program.cs
~~~
